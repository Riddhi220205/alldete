# Allergen Detector 

## Overview

This project detects food items using a Vision Transformer (ViT) model trained on the Food-101 dataset.

## Features

* Food image classification
* Allergen identification
* Pretrained ViT model support

## Setup


git clone <your-repo-url>
cd Allergen_detector
pip install -r requirements.txt


## Dataset

Download Food-101 dataset manually and place it in:

data/


## Model Weights

Download pretrained model and place in:

models/


## Run

python scripts/predict_food_item.py

Dataset: Available at https://drive.google.com/file/d/1xl78yeZwZW3N3rhuvW7WhuuA7lnKLvam/view?usp=drive_link
